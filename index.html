About the Workshop

Long-term memory is becoming a central capability for modern AI systems. While early language models were largely stateless and limited to a single context window, emerging assistants and agents increasingly store, retrieve, and update information across interactions, users, tasks, and modalities.

Such memory can enable more personalized assistants, long-horizon reasoning, task continuity, and richer multimodal understanding. At the same time, it raises fundamental questions: what should be remembered, how memories should be represented and retrieved, how systems should handle outdated or conflicting information, and how memory should interact with reasoning, planning, tool use, and user control.

PALM brings together researchers studying long-term memory from multiple perspectives — episodic and semantic memory, personalization, multimodal and visual memory, memory for LLM agents, cognitive and neuroscience-inspired models, and safety. The area remains fragmented across NLP, ML, agents, HCI, cognitive science, neuroscience, and AI safety; PALM aims to consolidate these threads, identify shared abstractions and evaluation protocols, and discuss the risks of memory architectures.

02

Topics of Interest

01

Memory Architectures for Conversational Agents

How should an assistant index, retrieve, read, and update information across sessions? Which abstractions generalize, and where do designs break down as histories grow?

02

Memory for LLM & Multi-Agent Systems

How should persistent experience, tool-use traces, and shared memory support long-horizon and collaborative tasks? When should memory be propagated, isolated, or provenance-tracked?

03

Personalization & Preference Memory

Which preferences should be retained, discarded, or updated, and how are conflicts resolved? How can we tailor without over-personalization, sycophancy, or repetition?

04

Multimodal, Visual, Video & Embodied Memory

How should memory be represented and retrieved across text, image, video, and sensorimotor streams, and evolved across long horizons with cross-modal dependencies?

05

Neuroscience-Inspired Memory Models

What can complementary learning systems, consolidation, and replay contribute? How should fast episodic encoding balance against slow semantic generalization and forgetting?

06

Safety, Privacy & Security of Persistent Memory

How can memory be protected against poisoning, injection, and sleeper attacks? How does persistent memory affect behavior over time — sycophancy, belief reinforcement, alignment drift?

07

User Control, Transparency & HXAI

How can users inspect, edit, and delete memories? What interfaces make memory architectures transparent and controllable?

Call for Papers

PALM welcomes work on long-term memory for AI systems. Accepted papers are non-archival — they may be submitted to or published at other venues.

Format

NeurIPS 2026 LaTeX style template, submitted as a PDF via OpenReview.

Length

Up to 4 pages, excluding references and appendix. (to be confirmed)

Review

Double-blind — please anonymize your submission. (to be confirmed)

Dual Submission

Allowed. As a non-archival workshop, PALM publishes no proceedings.

We welcome

Work in progress  ·  position papers  ·  benchmark proposals  ·  negative results  ·  early empirical studies.
